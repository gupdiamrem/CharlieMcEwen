Dungeon 1 Recreation of 1986 Version of The Legend of Zelda

A recreation of The Legend of Zelda for the SNES, built in C# as part of a team project for CSE 3902. The project includes gameplay mechanics, a save system, and various features reflective of the original game.

The project is under a private repository on my GitHub as required for now, but I am more than happy to grant access to it if requested by email.

Matrix Calculator Application

A personal project, inspired by my lack of success in finding a fully comprehensive matrix calculator while taking linear algebra. It is capable of computing matrix addition, subtraction, multiplication, transposition, the matrix inverse, and the determinant.

It is written in Java and utilizes the JavaFX framework for its GUI. I am currently implementing more features such as eigenvalue and vector computation, as well as teaching myself JavaScript to create a better interface for the application.
